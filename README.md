# 💬 Chat Project (Real-time Messenger)

Real-time SPA-мессенджер с синхронизацией сообщений через WebSocket.

## 🚀 Функционал

- ✅ Отправка и получение сообщений в реальном времени (WebSocket)
- ✅ Отображение истории сообщений
- ✅ Индикация статуса подключения (онлайн/оффлайн)
- ✅ Адаптивный интерфейс

## 🛠 Технологии

| Категория | Технологии |
|---|---|
| **Фронтенд** | React, TypeScript, JavaScript, HTML5, CSS3 |
| **Real-time** | WebSocket (нативное API) |
| **Управление состоянием** | React Hooks (useState, useEffect, useContext) |
| **Стили** | CSS Modules |
| **Сборка** | Create React App |
| **Бэкенд** | Node.js, Express, WebSocket (`ws`) |
