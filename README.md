# 💬 Chat Project (Real-time Messenger)

**Real-time SPA-мессенджер** с синхронизацией сообщений через WebSocket. Пользователи могут обмениваться сообщениями в реальном времени без перезагрузки страницы.

📦 **GitHub:** [Wjascheslaw88/4_Chat](https://github.com/Wjascheslaw88/4_Chat)

---

## 🚀 Функционал

- ✅ Отправка и получение сообщений в реальном времени (WebSocket)
- ✅ Отображение истории сообщений
- ✅ Подключение к чат-серверу с автоматическим переподключением
- ✅ Индикация статуса подключения (онлайн/оффлайн)
- ✅ Адаптивный интерфейс (мобильные устройства, планшеты, десктоп)
- ✅ Простая и понятная архитектура
